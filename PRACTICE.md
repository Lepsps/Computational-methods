### Билет 1. Построение интерполяционного полинома Лагранжа

**Источник:** Изображения №2, №4.

**Задача:**
Даны точки: $(-2, -8), (0, 2), (1, 4), (3, 32)$.
Построить интерполяционный полином Лагранжа $P_3(x)$ и привести его к каноническому виду.

**Решение:**
Полином ищется в виде:
$$P_3(x) = \sum_{i=1}^4 y_i L_i(x)$$
где $L_i(x)$ — базисные полиномы:
$$L_i(x) = \prod_{j \ne i} \frac{x - x_j}{x_i - x_j}$$

1.  **Вычисление базисных полиномов:**
    *   $x_1 = -2, y_1 = -8$:
        $$L_1(x) = \frac{(x-0)(x-1)(x-3)}{(-2-0)(-2-1)(-2-3)} = \frac{x^3 - 4x^2 + 3x}{-30}$$
    *   $x_2 = 0, y_2 = 2$:
        $$L_2(x) = \frac{(x+2)(x-1)(x-3)}{(0+2)(0-1)(0-3)} = \frac{x^3 - 2x^2 - 5x + 6}{6}$$
    *   $x_3 = 1, y_3 = 4$:
        $$L_3(x) = \frac{(x+2)x(x-3)}{(1+2)(1)(1-3)} = \frac{x^3 - x^2 - 6x}{-6}$$
    *   $x_4 = 3, y_4 = 32$:
        $$L_4(x) = \frac{(x+2)x(x-1)}{(3+2)(3)(3-1)} = \frac{x^3 + x^2 - 2x}{30}$$

2.  **Сборка полинома:**
    $$P(x) = -8 L_1(x) + 2 L_2(x) + 4 L_3(x) + 32 L_4(x)$$
    После приведения подобных слагаемых коэффициенты при $x^3, x^2, x$ суммируются.

**Ответ:** $P(x) = x^3 + x + 2$.

### Билет 2. Исследование задач на корректность

**Источник:** Изображения №2, №3, №4.

**Теория:** Задача корректна, если решение существует, единственно и устойчиво (малое изменение входа $\delta$ дает малое изменение выхода).

**Примеры из практики:**
1.  **Арифметическая прогрессия:** Найти $a_n = a_1 + (n-1)d$.
    Пусть $\tilde{a}_1 = a_1 + \delta$.
    Погрешность: $|\tilde{a}_n - a_n| = |\delta|$.
    **Вывод:** Задача корректна (ошибка не растет).

2.  **Геометрическая прогрессия:** Найти $b_n = b_1 \cdot q^{n-1}$.
    Пусть $\tilde{b}_1 = b_1 + \delta$, где $q > 1$.
    Погрешность: $|\tilde{b}_n - b_n| = |\delta \cdot q^{n-1}|$.
    При большом $n$ ошибка растет экспоненциально.
    **Вывод:** При $n \to \infty$ задача некорректна.

3.  **Линейное уравнение:** Найти корень $ax=b \Rightarrow x=b/a$.
    Пусть $\tilde{b} = b + \delta$.
    Погрешность: $|\tilde{x} - x| = |\delta / a|$.
    **Вывод:** Задача корректна (при $a \not\approx 0$).

4.  **Кратный корень полинома:** Уравнение $(x-a)^n = 0$.
    Возмущение свободного члена: $(x-a)^n = \delta$.
    Решение: $\tilde{x} = a + \sqrt[n]{\delta}$.
    Производная функции $\delta^{1/n}$ в нуле бесконечна. Малое $\delta$ дает ощутимое смещение корня.
    **Вывод:** Задача некорректна.

### Билет 3. Интерполяционный полином Ньютона

**Источник:** Изображения №5, №6.

**Задача:**
Даны точки: $(-2, 14), (0, 0), (1, 2), (3, 84), (6, 1302)$.
Построить интерполяционный полином в форме Ньютона.

**Решение:**
Строится таблица разделенных разностей $f(x_i, \dots, x_j)$:
1.  $f(x_i)$: $14, 0, 2, 84, 1302$.
2.  Разности 1-го порядка:
    $$\frac{0-14}{0-(-2)} = -7, \quad \frac{2-0}{1-0} = 2, \quad \dots$$
3.  Разности 2-го порядка:
    $$\frac{2-(-7)}{1-(-2)} = 3, \quad \frac{41-2}{3-0} = 13, \quad \dots$$
4.  Разности 3-го порядка:
    $$\frac{13-3}{3-(-2)} = 2, \quad \dots$$
5.  Разность 4-го порядка: $1$.

Формула Ньютона:
$$P(x) = f(x_0) + f(x_0,x_1)(x-x_0) + f(x_0,x_1,x_2)(x-x_0)(x-x_1) + \dots$$
Подстановка значений (верхняя строка таблицы):
$$P(x) = 14 - 7(x+2) + 3(x+2)x + 2(x+2)x(x-1) + 1(x+2)x(x-1)(x-3)$$

**Ответ:** После раскрытия скобок $P(x) = x^4 + x$.

### Билет 4. Метод наименьших квадратов (Линейная регрессия)

**Источник:** Изображения №6, №7.

**Задача:**
Аппроксимировать точки полиномом 1-го порядка ($y = ax+b$).
Точки: $(1, 1), (3, 7), (5, 1), (7, 7)$.

**Решение:**
Составляем систему нормальных уравнений:

$$\begin{cases} a \sum x_i^2 + b 
\sum x_i = \sum x_i y_i \\
a \sum x_i + b  
\cdot n = \sum y_i \end{cases}$$ 

Вычисляем суммы ($n=4$):
*   $\sum x_i = 1+3+5+7 = 16$
*   $\sum y_i = 1+7+1+7 = 16$
*   $\sum x_i^2 = 1+9+25+49 = 84$
*   $\sum x_i y_i = 1+21+5+49 = 76$

Получаем СЛАУ:

$$
\begin{cases} 
84a + 16b = 76 \\ 
16a + 4b = 16 
\end{cases}
$$

Второе уравнение делим на 4: $4a + b = 4 \Rightarrow b = 4 - 4a$.
Подставляем в первое: $84a + 16(4-4a) = 76 \Rightarrow 20a = 12 \Rightarrow a = 0.6$.
Тогда $b = 4 - 2.4 = 1.6$.

**Ответ:** Аппроксимирующая прямая $f(x) = 0.6x + 1.6$.

### Билет 5. Построение кубического сплайна

**Источник:** Изображения №8, №9.

**Задача:**
Построить сплайн для точек $(1, 1), (3, 7), (5, 1), (7, 7)$ с естественными граничными условиями ($\sigma_1=\sigma_2=0$).

**Решение:**
Шаги сетки $h_i = 2$.
Обозначим $C_i$ — вторые производные в узлах. Из условий $C_1=C_4=0$ (в тетради нумерация может быть сдвинута, но суть та же).
Основная система для внутренних узлов:
$$h_i C_{i-1} + 2(h_i + h_{i+1})C_i + h_{i+1}C_{i+1} = 6 \left( \frac{y_{i+1}-y_i}{h_{i+1}} - \frac{y_i-y_{i-1}}{h_i} \right)$$
Подстановка данных дает систему для $C_2, C_3$:

  $$\begin{cases} 
8C_2 + 2C_3 = -18 \\ 
2C_2 + 8C_3 = 18 
\end{cases}$$  

Решение системы: $C_2 = -3, C_3 = 3$.
Сплайн на каждом отрезке $i$ строится по формуле:


  $$S_i(x) = a_i + b_i(x-x_i) + \frac{C_i}{2}(x-x_i)^2 + \frac{d_i}{6}(x-x_i)^3$$  


Коэффициенты вычисляются через $y_i$ и найденные $C_i$.

### Билет 6. Полиномы Чебышёва

**Источник:** Изображения №9, №10.

**Задача:**
1. Построить полином Чебышёва $T_5(x)$.
2. Перенести (масштабировать) его на отрезок $[1, 7]$.

**Решение:**
1.  **Рекуррентная формула:**
    $T_n(x) = 2x T_{n-1}(x) - T_{n-2}(x)$; $T_0=1, T_1=x$.
    *   $T_2 = 2x^2 - 1$
    *   $T_3 = 4x^3 - 3x$
    *   $T_4 = 8x^4 - 8x^2 + 1$
    *   $T_5 = 16x^5 - 20x^3 + 5x$
    Нормированный (со старшим коэффициентом 1): $\tilde{T}_5 = x^5 - \frac{5}{4}x^3 + \frac{5}{16}x$.

2.  **Смена интервала:**
    Отображение $t \in [1, 7] \to x \in [-1, 1]$.
    $$t = \frac{a+b}{2} + \frac{b-a}{2}x = 4 + 3x \implies x = \frac{t-4}{3}$$
    Искомый полином на $[1, 7]$:
    $$\hat{T}_5(t) = \tilde{T}_5\left(\frac{t-4}{3}\right)$$
    Этот полином имеет наименьшее отклонение от нуля на отрезке $[1, 7]$.

### Билет 7. Решение СЛАУ методом итераций

**Источник:** Изображения №11, №12.

**Задача:**
Решить систему:

$$ 
\begin{cases} 
4x_1 + 2x_2 + x_3 = 7 \\ 
x_1 + 4x_2 - x_3 = 4 \\ 
x_1 + x_2 + x_3 = 3 
\end{cases} 
$$

**Решение:**
1.  **Проверка сходимости (диагональное преобладание):**
    *   Строка 1: $|4| > |2| + |1|$ (ОК).
    *   Строка 2: $|4| > |1| + |-1|$ (ОК).
    *   Строка 3: $|1| < |1| + |1|$ (Нет преобладания).
2.  **Преобразование:**
    Выполняем линейные операции со строками, чтобы добиться $|a_{33}| > |a_{31}| + |a_{32}|$.
    В тетради показан подбор коэффициентов $\alpha, \beta$ для линейной комбинации уравнений.
3.  **Итерационный процесс:**
    Приводим к виду $x = Bx + c$.
    $x_1^{(k+1)} = \frac{1}{4}(7 - 2x_2^{(k)} - x_3^{(k)})$ и т.д.
    Вычисляем последовательные приближения $X_0, X_1, \dots$.
    Погрешность $\varepsilon_k = ||X_k - X_{k-1}||$ уменьшается с каждым шагом.

### Билет 8. Вывод квадратурных формул Ньютона-Котеса

**Источник:** Изображения №12, №13, №14.

**Задача:**
Найти веса квадратурной формулы для 5 равноотстоящих точек на отрезке $[-1, 1]$: $x_i \in \{-1, -0.5, 0, 0.5, 1\}$.
$$\int_{-1}^1 f(x) dx \approx \sum_{i=0}^4 w_i f(x_i)$$

**Решение (Метод неопределенных коэффициентов):**
Требуем, чтобы формула была точна для $1, x, x^2, x^3, x^4$.
Система уравнений:
1.  $\sum w_i = \int_{-1}^1 1 dx = 2$
2.  $\sum w_i x_i = 0$ (из нечетности $x$)
3.  $\sum w_i x_i^2 = \int_{-1}^1 x^2 dx = \frac{2}{3}$
4.  $\sum w_i x_i^3 = 0$
5.  $\sum w_i x_i^4 = \frac{2}{5}$

Из симметрии $w_0=w_4, w_1=w_3$. Решаем упрощенную систему матричным методом (Гаусса).
Результат (веса): $w_2 = \frac{12}{90}, w_1 = w_3 = \frac{32}{90}, w_0 = w_4 = \frac{7}{90}$.

### Билет 9. Построение формул Гаусса (n=2)

**Источник:** Изображение №15.

**Задача:**
Найти узлы и веса для формулы Гаусса с 2 узлами ($n=2$) на $[-1, 1]$.
$$I \approx c_1 f(\xi_1) + c_2 f(\xi_2)$$

**Решение:**
Система уравнений для моментов до 3-го порядка ($2n-1 = 3$):
1.  $c_1 + c_2 = 2$
2.  $c_1 \xi_1 + c_2 \xi_2 = 0$
3.  $c_1 \xi_1^2 + c_2 \xi_2^2 = 2/3$
4.  $c_1 \xi_1^3 + c_2 \xi_2^3 = 0$

Из симметрии задачи: $\xi_1 = -\xi_2$ и $c_1 = c_2$.
Из (1): $2c_1 = 2 \Rightarrow c_1 = c_2 = 1$.
Из (3): $1 \cdot \xi_1^2 + 1 \cdot (-\xi_1)^2 = 2/3 \Rightarrow 2\xi_1^2 = 2/3 \Rightarrow \xi_1 = -1/\sqrt{3}, \xi_2 = 1/\sqrt{3}$.

**Ответ:** Узлы $\pm \frac{1}{\sqrt{3}}$, веса $1$.

### Билет 10. Построение формул Гаусса (n=3)

**Источник:** Изображения №15, №16.

**Задача:**
Найти узлы и веса для формулы Гаусса с 3 узлами.

**Решение:**
Система уравнений включает моменты до $x^5$ ($\int x^k dx$).
$$c_1 \xi_1^k + c_2 \xi_2^k + c_3 \xi_3^k = \int_{-1}^1 x^k dx$$
Используем симметрию:
Центральный узел $\xi_2 = 0$. Симметричные узлы $\xi_1 = -\xi_3$. Веса $c_1 = c_3$.
Система упрощается, остаются уравнения для $c_1, c_2, \xi_3$:
1.  $2c_1 + c_2 = 2$
2.  $2c_1 \xi_3^2 = 2/3$
3.  $2c_1 \xi_3^4 = 2/5$

Делим (3) на (2): $\xi_3^2 = \frac{2/5}{2/3} = \frac{3}{5} \Rightarrow \xi_3 = \sqrt{0.6}$.
Находим веса: $c_1 = \frac{5}{9}$, тогда $c_2 = 2 - 10/9 = 8/9$.

**Ответ:** Узлы $0, \pm \sqrt{0.6}$. Веса: $8/9$ (центр), $5/9$ (края).
